# Образовательный тамагочи

## Идея и ценность

### Проблема

Дети 7–11 лет проводят много времени за экранами, но редко используют это время продуктивно. Родителям сложно мотивировать ребёнка читать, рисовать или заниматься физкультурой — особенно когда конкурируют игры и видео.

### Решение

Образовательный тамагочи — это образовательная игра в жанре тамагочи, где виртуальный питомец отражает реальные активности ребёнка. Хочешь чтобы питомец был счастливым и здоровым? Читай книги, рисуй и делай зарядку.

### Почему это круто

- **Геймификация обучения** — ребёнок получает опыт и повышает уровень питомца за реальные полезные действия
- **Родительский контроль** — взрослый подтверждает выполнение активностей через пинкод или математическую задачу
- **Чтение с квизом** — после чтения ребёнок проходит тест на понимание текста
- **Рисование** — ребёнок выбирает тему и рисует под таймер
- **Спорт** — набор упражнений с таймером и передышками между ними
- **Прогресс виден** — шкалы нужд и уровень питомца отображают активность ребёнка в реальном времени

### Взаимодействие компонентов

**React ↔ Бэкенд** — через REST API:

- Авторизация и регистрация родителя
- Управление refresh токеном через httpOnly cookie

**Unity ↔ Бэкенд** — через REST API:

- Загрузка профиля ребёнка и состояния питомца
- Автосохранение состояния каждые 30 секунд
- Отправка активностей (чтение, рисование, спорт)

**React ↔ Unity** — через JavaScript bridge (`SendMessage`):

- React передаёт access токен в Unity после логина
- Unity запрашивает refresh через React при истечении токена
- React отправляет обновлённый токен обратно в Unity

## 🗄 База данных

```sql
users              — аккаунты родителей (email, password_hash)
child_profiles     — профили детей (name, gender, parent_pin, has_pet)
pets               — питомцы (type, gender, level, xp, needs)
activities         — история активностей (type, confirmed_by_parent)
```

## 🚀 Как запустить проект

### Требования

- Node.js 22.12+
- Go 1.21+
- PostgreSQL 15+
- Unity 2022.3+ (для разработки игры)

---

### 1. Бэкенд и бд

```bash
# Клонировать репозиторий
git clone https://github.com/scmbr/vk-gamejam

```

```bash
# Запуск контейнера
docker compose -p vk-gamejam up --build
```

---

### 2. Фронтенд

```bash
cd vk-gamejam/frontend

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
# Фронтенд запустится на http://localhost:3000
```

## 📡 API

| Метод | Эндпоинт             | Описание                 |
| ----- | -------------------- | ------------------------ |
| POST  | `/api/auth/register` | Регистрация              |
| POST  | `/api/auth/login`    | Авторизация              |
| POST  | `/api/auth/refresh`  | Обновление токена        |
| POST  | `/api/auth/logout`   | Выход                    |
| GET   | `/api/users/me`      | Данные аккаунта          |
| POST  | `/api/child/profile` | Создание профиля ребёнка |

| PUT   | `/api/child/profile` | Обновление профиля       |
